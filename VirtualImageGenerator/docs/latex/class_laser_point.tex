\doxysubsection{Detailed Description}
Represents a 3D point with associated ID and color. 

This class stores a 3D point\textquotesingle{}s coordinates (in the form of a dynamic array), a unique ID for the point, and a color represented by an RGB triplet (red, green, blue). It handles memory management for the dynamic array storing the 3D coordinates.
