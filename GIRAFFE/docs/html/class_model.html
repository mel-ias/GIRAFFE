Exports a recolored point cloud to a text file with optional coordinate shifting.

This function saves the 3D points of a point cloud along with their associated colors and image coordinates to a text file. The coordinates can be shifted by specified amounts along the X, Y, and Z axes. Only points with non-zero colors are included in the output.

Parameters
workingDirectoryThe directory where the output file will be saved.
wD_nameThe base name for the output file (without extension).
point_cloudA vector of 3D points representing the point cloud.
point_cloud_colorsA vector of colors corresponding to each point in the point cloud (in BGR format).
image_coords_colorsA vector of 2D image coordinates corresponding to each point in the point cloud.
shifter_xThe amount to shift the X coordinates of the points.
shifter_yThe amount to shift the Y coordinates of the points.
shifter_zThe amount to shift the Z coordinates of the points.

const std::vector< cv::Point2d > & image_pixels )

Finds the nearest neighbor indices in image_pixels for each point in image_points.

Uses a KD-tree to efficiently locate the nearest neighbor in image_pixels for each 2D point in image_points, based on Euclidean distance. The KD-tree is constructed using FLANN.

Parameters
image_pointsVector of 2D points for which nearest neighbors are sought.
image_pixelsVector of 2D points representing the search space.
Returns
A vector of indices, each corresponding to the closest point in image_pixels for each point in image_points.

Projects a point cloud onto an image and retrieves colors from the image.

This function projects 3D points from a given point cloud into 2D image coordinates using the provided camera parameters. It then extracts the color values from the specified image for the projected points and associates them with a list of image points.

Parameters
point_cloudA vector of 3D points representing the point cloud.
image_for_colorThe image from which color values will be extracted.
point_cloud_colorsA reference to a vector where the colors corresponding to the projected points will be stored.
image_coords_colorsA reference to a vector where the valid image coordinates of the projected points will be stored.
fix_aspect_ratioA boolean flag indicating whether to maintain the aspect ratio during the projection.
cameraMatrixThe intrinsic camera matrix used for projecting the points.
distCoeffsThe distortion coefficients for correcting the image.
rvecThe rotation vector for the camera pose.
tvecThe translation vector for the camera pose.
image_pointsA vector of 2D points representing the image points to be matched with the projected points.
Returns
A vector of pairs, where each pair contains a projected 3D point and the index of the corresponding image point.
